Textbooks

The official textbook for this course is

  • Operating Systems: Three easy pieces by Remzi H. Arpaci-Dusseau and Andrea C. Arpaci-Dusseau.
As further background, we suggest
  • Principles of Computer System Design, by Jerome Saltzer and Frans Kaashoek. You can read this book online by logging into the Rutgers library site; use this custom link. Also, Part 2 of this book is freely available online.
  • The Linux Programming Interface : A Linux and UNIX System Programming Handbook, by Michael Kerrisk. You can read this book online by logging into the Rutgers library site; use this custom link.
  • Advanced programming in the Unix environment by W. Richard Stevens and Stephen A. Rago.

The instructor may provide additional reading materials as and where needed for each lecture's topics.


Syllabus

This syllabus is tentative and subject to change.
  • CPU virtualization: processes, scheduling
  • Memory virtualization: address spaces, translation, segmentation, paging, swapping
  • Concurrency: threading, locks, condition variables, semaphores
  • Persistence: RAID, file systems, journaling
  • (only if time permits) Security: authentication, access control, cryptography
